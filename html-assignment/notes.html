My Notes:

5 HTML tags I have learnt and their description

img
Links images to the pages. "src" attribute specifies the path to the image
a
defines a hyperlink. "href" attribute is the most important elements of "a" tag
ol and ul
They are used to list items (ordered and unordered respectively). "li" attribute is used for each element
blockquote
used to indicate the quotation of a large section of text from another source
p
It represents a paragraph
